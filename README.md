## SpringBoot_Blog
스프링부트를 학습하며 구현한 블로그

### 개발기간
24.02.20 - 24.05.17

### 개발환경
- JAVA 17 (with JS, JSP)
- Tool: STS4(Eclipse)
- Framework: SpringBoot (3.x)
- DB: MySQL
- ORM: JPA

<br>

### 주요기능
**회원가입 및 로그인**
- Spring Security
- 비밀번호 암호화(Hash)
- OAuth 2.0을 통한 SNS 회원가입 및 로그인

**마이페이지**
- 회원정보 확인 및 수정

**글 및 댓글**
- 권한있는 사용자 글과 댓글 작성
- 작성자만이 글과 댓글 삭제

**메인페이지**
- 최신글 상단 위치
